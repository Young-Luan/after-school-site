# 课后复习网站完整流程（按你确认的顺序）

## 第1阶段：纯内容网站（当前阶段）
目标：不显示登录，先把网站内容搭好并反复修改。

你现在只需要做：
1. 打开 `index.html` 看页面效果。
2. 把你想改的栏目/文案/题目发给我，我直接改。

当前状态：已完成。

## 第2阶段：上线公开访问（先不做登录）
目标：让家长和学生能在公网访问同一个网址。

执行内容：
1. 用 Vercel 部署静态站点。
2. 验证手机和电脑都能打开。

你需要提供：
1. GitHub 账号（用于托管代码）。
2. Vercel 账号（用 GitHub 登录即可）。

## 第3阶段：账号系统（管理员分发账号）
目标：你创建账号，学员和家长只登录，不自助注册。

执行内容：
1. 接入 Supabase Auth。
2. 关闭公开注册。
3. 你创建 admin / parent / student 账号。
4. 首次登录强制改密码。

说明：
- 相关基础文件我已预留在 `supabase/` 目录中。
- 这一阶段开始前，我会再把页面切换到登录版本。

## 第4阶段：学习数据上云
目标：学习记录不再只保存在本地浏览器，换设备也能同步。

执行内容：
1. 把打卡、学习时长、练习结果存入 Supabase 数据库。
2. 按账号读取个人数据。
3. 管理员可查看班级/学员进度。

## 第5阶段：题库可配置（非程序员可维护）
目标：你不用改代码也能改题目。

执行内容：
1. 支持 CSV/表格导入题库。
2. 题库版本管理（防止误覆盖）。
3. 导入后自动生效。
